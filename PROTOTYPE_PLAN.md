# 🎯 Prototype Zero（执行清单版）

## 🎯 目标
- 做出一个能连续打10分钟的战斗循环

---

## 🔁 核心循环
- 移动 → 攻击 → 命中 → 击杀 → 重复

---

## 🧱 执行步骤（按顺序）

### 1️⃣ 素材
- [ ] 找1个玩家sprite（人形，含idle+attack）
- [ ] 找2个敌人sprite（慢厚 / 快脆）
- [ ] 导入Godot（AnimatedSprite2D）
- [ ] 动画能正常播放

---

### 2️⃣ 玩家
- [ ] WASD移动
- [ ] 攻击（动画 + hitbox）
- [ ] 血量系统
- [ ] 死亡

---

### 3️⃣ 敌人（2种）
- [ ] 慢厚：慢速 / 高血
- [ ] 快脆：快速 / 低血
- [ ] 追玩家
- [ ] 可被击杀

---

### 4️⃣ 战斗
- [ ] 命中判定（hitbox）
- [ ] 扣血
- [ ] 敌人死亡消失

---

### 5️⃣ 反馈
- [ ] 命中反馈（闪 / 抖 / 停顿）
- [ ] 攻击有节奏（前摇/停顿）
- [ ] 受伤有反应
- [ ] 死亡清晰

---

### 6️⃣ 循环
- [ ] 敌人自动生成
- [ ] 可持续战斗
- [ ] 死亡后可重开

---

## 🧠 验收标准
- [ ] 打同一只怪20次不烦
